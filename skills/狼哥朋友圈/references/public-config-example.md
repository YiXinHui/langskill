# 狼哥朋友圈 个人配置模板

> 克隆后复制本文件为 `references/personal-config.md`，填入你的路径。

## 占位符映射

| 占位符 | 你的路径 |
|-------|---------|
| `<朋友圈框架文件>` | `<你的朋友圈方法论 md 文件>.md` — 这是 skill 的核心操作手册（常量/类型定义/写作规则） |
| `<会话日志根>` | `<你的会话日志归档目录>/` |
| `<录音分析水库>` | `<你的录音分析目录>/` |
| `<活动复盘目录>` | `<你的活动/直播复盘目录>/` |
| `<客户档案目录>` | `<你的客户档案目录>/` |
| `<知识单元库>` | `<你的知识单元/卡片目录>/` |
| `<金句库>` | `<你的金句库目录>/` |
| `<素材库根>` | `<你的本地素材库根目录>/` |
| `<朋友圈输出目录>` | `<你的朋友圈每日规划输出目录>/` |
