# langgeladi — 狼格拉底AI商业工具箱

帮中小企业老板和AI咨询顾问做三件事：诊断客户、定产品、做AI化。

## 安装

```bash
npx skills add YiXinHui/langskill
```

## 包含的工具

| 命令 | 工具 | 说明 |
|------|------|------|
| `/lang` | 路由入口 | 自动分发到最合适的诊断工具 |
| `/lang-diagnosis` | 商业诊断 | 问诊/体检两种模式，消解客户伪需求 |
| `/lang-think` | 业务AI化 | TEKI六阶段，把隐性知识变成AI流程 |
| `/lang-mirror` | 老板自检 | AI就绪度五级评估（即将推出） |
| `/lang-positioning` | 产品定位 | 四层模型 + 产品包装六原则（即将推出） |
| `/lang-traffic` | 流量获客 | 四行业类型模型 + 内容即资产（即将推出） |

## 核心理念

- **人只会为自己得出的结论买单** — 诊断不是给答案，是帮人问对问题
- **私有方法论 > 公开AI智能** — AI是放大器，放大你的强项也放大你的弱项
- **AI系统一定是长出来的** — 先打穿一个点，别想一步到位
